### Set-up

1. pipenv
    - virtual envrionment with including Python packages
    - can easily manage different packages
    - setting up:
        - install pipenv
        ```
        pip3 install pipenv
        ```
        - set the directory to be the pipenv virtual environment
        ```
        pipenv install
        ```
        - launch into the virtual environment
        ```
        pipenv shell
        ```
        - get out of pipenv
        ```
        exit
        ```
